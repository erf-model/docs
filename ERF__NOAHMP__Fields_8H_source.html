4 // ===========================================================================
5 // ERF <-> Noah-MP coupling contract
6 // ---------------------------------------------------------------------------
7 // This header is the single place that declares WHAT crosses the ERF/Noah-MP
8 // boundary. Four (well, five) enums describe TWO different boundaries; see
9 // dev/spec-noahmp-api.md and dev/spec-noahmp-reorg.md.
10 //
11 // - LsmData_NOAHMP / LsmFlux_NOAHMP : ERF-internal coupling fields, read
12 // and written by the rest of ERF (radiation, surface layer). Persisted to
13 // the LSM plotfile / checkpoint BY NAME.
14 // - NoahmpInputComp / NoahmpOutputComp : transient per-step staging-buffer
15 // components used only inside Advance_With_State. Never persisted.
16 // - NoahmpPrecipSlot : the typed precip accumulator slots.
17 //
18 // Mechanical (pure 1:1) coupled fields are declared ONCE below as X-macro
19 // lists (NOAHMP_*_FIELDS). Expanding a list with different X definitions
20 // generates the enum entries, the name strings, and the host<->NoahmpIO copy
21 // lines from one source of truth, so they cannot drift. Fields that carry
22 // per-field math (winds, EOS, precip, banded albedos, fluxes, soil) are NOT
23 // table-driven and stay explicit in the .cpp -- see the "NOT table-driven"
24 // note in dev/spec-noahmp-reorg.md §4.
25 // ===========================================================================
